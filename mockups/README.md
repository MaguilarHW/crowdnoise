## mockups (dark mode)

these are lightweight, [untitled]-ish concept screens for **crowd·noise** based on `idea.md` + `whatitlookslike.md`.

### screens
- **home**: `mockups/home.svg`
- **sample editing**: `mockups/sample_editing.svg`
- **provenance / original sound**: `mockups/provenance_original.svg`
- **voting (creativity)**: `mockups/voting_creativity.svg`

### style notes
- dark background, white text, soft-gray dividers
- lowercase labels, minimal chrome, big whitespace
- “one primary action” per screen

### regenerate

```bash
python3 mockups/generate_mockups.py
```

# crowd·noise website mockup

A mobile-first, dark-mode website mockup for screenshots and demos.

## how to use

1. **Open locally**: Just open `index.html` in your browser (double-click or `open index.html`)

2. **Take screenshots**: 
   - The site is designed to look like a mobile app
   - Use browser dev tools (F12) to set mobile viewport if needed
   - Or resize your browser window to ~390px wide

3. **Navigate**: Click cards and buttons to switch between screens:
   - **Home** → shows groups and active projects
   - **Editing** → sample editing with sliders
   - **Provenance** → original sound playback and transformation chain
   - **Voting** → creativity rating interface

## screens included

- **home**: groups + active projects
- **editing**: minimal sliders for sound transformation
- **provenance**: visual chain showing where sounds came from
- **voting**: creativity rating with stars

## styling

- Dark mode (`#0b0b0d` background)
- [untitled]-style minimalism
- Lowercase everywhere
- Mobile-first responsive
- Smooth transitions between screens

## notes

- No server needed — just static HTML/CSS/JS
- Interactive elements work (sliders, stars, play buttons)
- Perfect for screenshots and presentations
